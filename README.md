# だいちの木 ピアノ

放課後等デイ「だいちの木/だいち」の発達特性児が、**意志でなく仕組みで**ピアノを学ぶための、完全無料・サーバ不要・iPad mini(iOS Safari/PWA)アプリ。依存ライブラリゼロ（自前SVG楽譜＋Web Audio）。

## 中核思想
**同じ楽譜に「足場」を重ね、その濃さを子ごと・手ごと独立に増減し、慣れたら剥がす。** 誤りなし（失敗音ゼロ）＋ご褒美で運用。読める子も読めない子も同じ画面で、それぞれのちょうどいい足場で弾ける。

## できること
- **手ごと独立の読みオンオフ**: 音名(ドレミ)/色/指番号 を「みぎて・ひだりて」別々にON/OFF。「左だけ読みオン」も当然できる。
- **一小節モード**: 1小節だけ大きく表示、◀▶で移動、くりかえし。長い曲で迷子にならない。
- **3つのひきかた**:
  - **おてほん** = アプリが全部弾く（聴いて覚える）
  - **いっしょ** = 練習しない手を伴奏、テンポで進む
  - **まつ** = 正しい音を弾くまで待って1音ずつ進む（**ゆっくりでも初見で弾ける**核）。電子キーボードの音をマイクで拾うか、画面鍵盤タップ、「つぎ」ボタンで前進。
- **光ナビ（落下ノーツ）**: 押す鍵盤が音の色で光る。読譜なしでも弾ける（たいち向け）。
- **色音符**: ド=赤 レ=橙 ミ=黄 ファ=緑 ソ=青 ラ=紫 シ=桃（Boomwhackers系）。色＋五線の高さ＋音名の三重で、色を消しても読める。
- **児童プロファイル**: わかな/みこと/たいち/かほ をタップで、その子に合う足場・モード・テンポ・ご褒美に一発切替。
- **ご褒美**: できた演出＋星。しずか/ふつう/にぎやか で過刺激を調整。失敗音は鳴らない。
- **曲の取り込み**: 著作権曲（さんぽ等）は「だいち記譜」で先生が打てば端末内だけに保存（配布物に含めない）。

## 使い方（先生）
1. iPad/iPhoneのSafariでURLを開く →「ホーム画面に追加」で全画面アプリに。
2. **先生メニュー** = 画面**右上の「⚙ せんせい」ボタン**（PCでは `g` キーや右上角の長押しでも開く）。
3. メニューで こども選択 → 曲 → ひきかた・足場・ご褒美 を設定。
4. 子ども用バー（下）の大ボタンで ▶再生 / つぎ / **🎤マイク** / みぎ・ひだり・りょうて / ぜんたい・1しょうせつ。
5. **まつモードで電子キーボードの音で進めたい時は 🎤 をタップ**（赤く光れば聴いている）。マイクなしでも 🎹画面鍵盤タップ や つぎ で進む。
6. iOSの**アクセスガイド**併用推奨（誤操作・離脱防止）。

> 一小節モードは「表示を1小節に拡大」するだけで、再生・まつは曲全体を進み、表示が自動で追いかける（曲は冒頭だけではない。下の「1 / 12」が全体の小節数）。

## だいち記譜（取り込みフォーマット）
```
title: きょくめい
key: C
time: 4/4
tempo: 90
RH: C4q D4q E4q C4q | E4h G4h
LH: [C3G3]h [C3G3]h | [G2D3]h [C3G3]h
```
- 音価: `w`=全 `h`=2分 `q`=4分 `e`=8分 `s`=16分、付点は末尾 `.`（`h.`=3拍）
- 和音: `[C3E3G3]` / 休符: `R` / 指番号: `C4q:3` / 小節区切り: `|`（任意）
- 音名+オクターブ: 中央ド=`C4`（MIDI60）。`#`/`b` 可。

## 内蔵曲（パブリックドメイン・メロディのみ・歌詞非収録）
きらきら星 / メリーさんのひつじ / ちょうちょう / かえるのうた / 喜びの歌 / ハッピーバースデー（原語メロディ・2016和解でPD確定）。
右手=単音旋律、左手=各小節2音（根音+5度）。

## デプロイ（GitHub Pages・無料）

## 技術メモ
- 依存ゼロ。`js/` は theory(音名/色/五線) / songs(記譜パーサ+曲) / audio(合成ピアノ) / pitch(マイク単音検出) / render(SVG楽譜) / keyboard(鍵盤+落下) / app(制御)。
- **MIDIは非実装**（iOS SafariがWeb MIDI非対応のため）。入力はマイク単音検出（autocorrelation）＋画面タッチ。和音採点は不可なので「鳴ればOK」。
- Service Workerでオフライン動作（レッスンWi-Fi不安定でも落ちない）。取り込み曲はlocalStorage。
- 画像はゼロでも成立（楽譜・鍵盤・色は全てSVG/CSS描画）。アイコンはcodexで差し替え予定 → `assets/CODEX_ASSETS.md`。

## 既知の制約
- マイク採点は単音のみ・安価キーボードや声では精度が落ちる。
- 色値はBoomwhackers近似（公式HEXは存在しない）。Figurenotesとは別系統。
- 内蔵曲は権利クリアなPDのみ。流行曲は先生が取り込みで（ローカル保存）。
